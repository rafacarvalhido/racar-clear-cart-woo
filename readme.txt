RaCar Clear Cart WooCommerce allows you to add a customizable button to clear the cart.

== Description ==

RaCar Clear Cart WooCommerce is a free eCommerce plugin that allows you to remove all items from you shopping cart with 2 clicks (1 to send action and another to confirm.

== Installation ==

= Minimum Requirements =

* PHP version 5.6 or greater (PHP 7.2 or greater is recommended)
* MySQL version 5.6 or greater 


= Automatic installation =

Automatic installation is the easiest option as WordPress handles the file transfers itself and you don’t need to leave your web browser. To do an automatic install of RaCar Clear Cart WooCommerce, log in to your WordPress dashboard, navigate to the Plugins menu and click Add New.

In the search field type “RaCar Clear Cart WooCommerce” and click Search Plugins. Once you’ve found our plugin you can view details about it such as the point release, rating and description. Most importantly of course, you can install it by simply clicking “Install Now”.

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
